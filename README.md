# Bipartite Graph Movie Recommendation System
## Overview

This project implements a simple movie recommendation system using a bipartite graph structure. Users and movies are represented as two different node sets, and connections between them are created based on user ratings.

The main idea is to find similarities between users through graph traversal and recommend movies accordingly.

## Features
Bipartite graph structure (users and movies)  
Rating-based edge weights (1 / rating)  
Shortest path calculation between users  
Basic recommendation logic based on similar users

## Project Structure

/src → source files  
/data → rating dataset  
main.cpp → main file
